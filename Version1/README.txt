    O projeto foi realizado em Python, para poder executar o programa deverá
        ter instalado as bibliotecas numpy,matplotlib,csv,pprint,prettytable,
        palettable e mpl_toolkits, para as instalar poderá fazer:
            pip install <biblioteca>

    Para executar o programa principal, execute o comando: python Main.py

    Nota: o ficheiro VisualizeNN.py não é da nossa autoria, foi retirado de
        https://github.com/jzliu-100/visualize-neural-network sendo o seu
        autor Jianzheng Liu, este ficheiro é de licensa pública, apenas o 
        utilizamos para representar a rede neuronal (o desenho dos nós).
    

    Estes programas foram criados e executados em máquina  64 bits Windows 10 
        Home com 16 GB de RAM, Intel Core i7-8550U CPU @ 1.80GHz 1.99 GHz 8 cores


    As versões de Python utilizada foi: Python 3.6.8
